三、CNN 模型结构
本次实验采用轻量级卷积神经网络，结构如下：
表格
层类型	输出尺寸	功能说明
输入层	1×28×28	接收单通道 28×28 手写数字图像
卷积层 1	16×26×26	16 个 3×3 卷积核，步长 1，提取低级边缘特征
池化层 1	16×13×13	2×2 最大池化，步长 2，降低特征图尺寸，减少计算量
卷积层 2	32×11×11	32 个 3×3 卷积核，步长 1，提取高级组合特征
池化层 2	32×5×5	2×2 最大池化，步长 2，进一步压缩特征维度
全连接层 1	128	将特征图展平后映射为 128 维特征向量
全连接层 2	10	输出 10 维向量，对应 0-9 共 10 个数字类别
模型结构代码：
python
class Net(nn.Module):
    def __init__(self):
        super().__init__()
        self.conv1 = nn.Conv2d(1, 16, 3)
        self.pool = nn.MaxPool2d(2, 2)
        self.conv2 = nn.Conv2d(16, 32, 3)
        self.fc1 = nn.Linear(32 * 5 * 5, 128)
        self.fc2 = nn.Linear(128, 10)

    def forward(self, x):
        x = self.pool(F.relu(self.conv1(x)))
        x = self.pool(F.relu(self.conv2(x)))
        x = torch.flatten(x, 1)
        x = F.relu(self.fc1(x))
        x = self.fc2(x)
        return x
